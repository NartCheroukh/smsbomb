# Python SMS Request Testing Tool

A Python-based SMS request testing tool developed for educational and research purposes.
This project demonstrates how automated HTTP requests, multithreading, and API interactions can be used to simulate SMS verification request flows commonly used by online services.

The tool allows users to input a phone number and programmatically send multiple SMS verification requests through different service endpoints. It utilizes Python's threading capabilities to perform multiple requests efficiently and test how systems handle repeated verification requests.

## Features

* Command-line interface
* Automated HTTP request handling
* Multithreaded request execution
* Modular service request functions
* Basic input validation
* Simple dependency setup

## Requirements

* Python 3.x
* requests
* colorama

Install dependencies using:

pip install -r requirements.txt

## Usage

Run the main script and follow the instructions in the terminal to enter the target phone number and select the desired options.
